Using UI components in HTML5

Example: lf-checklist

In the example below, the implementation comes from the CDN.

  1. Add the following script tags to index.html 

     <script src="https://cdn.jsdelivr.net/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-ui-components.js" defer></script>

  2. Choose a theme to style your components: 

     <link href="https://cdn.jsdelivr.net/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-laserfiche-lite.css" rel="stylesheet" />

    or 

     <link href="https://cdn.jsdelivr.net/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-ms-ofice-lite.css" rel="stylesheet" />

  3. Add the Angular Material stylesheet 

     <link href="https://cdn.jsdelivr.net/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/indigo-pink.css" rel="stylesheet" />

  4. Add Zone.js if not already imported 

         <script src="https://cdn.jsdelivr.net/npm/zone.js@0.11.4/bundles/zone.umd.min.js"></script>

  5. Add the following script tags to index.html 

    <lf-checklist id="myChecklist" ></lf-checklist>

  6. In the .js file (e.g.: app.js),

    const checklist = document.getElementById('mychecklist'); // create a reference for custom element
checklist.initAsync(...);

function onChecklistchanged(event) {
  const checklistItem = event.detail; // event data will be stored in the detail property
}

// we have to add event listener to listen for the HTML custom event checklistChanged
checklist.addEventListener('checklistChanged', onChecklistchanged);

  7. Add the .js script to index.html 

     <script src="./app.js" defer></script>
