Using UI Components from CDN in HTML5

Example: lf-checklist

In the example below, the implementation comes from the CDN.

  1. Follow these steps to get started.

  2. Add the following script tags to index.html 

    <lf-checklist id="myChecklist" ></lf-checklist>

  3. In the .js file (e.g.: app.js),

    const checklist = document.getElementById('mychecklist'); // create a reference for custom element
checklist.initAsync(...);

function onChecklistchanged(event) {
  const checklistItem = event.detail; // event data will be stored in the detail property
}

// we have to add event listener to listen for the HTML custom event checklistChanged
checklist.addEventListener('checklistChanged', onChecklistchanged);

  4. Add the .js script to index.html 

     <script src="./app.js" defer></script>
