Overview

Custom Element that displays a list of breadcrumbs, allowing the user to select the parent, or one from the dropdown.

HTML Element

Attribute Value Description
breadcrumbs LfBreadcrumb[] List of breadcrumbs to display

Event Value Description
breadcrumbClicked () => event: CustomEvent<{ selected: LfBreadcrumb; breadcrumbs: LfBreadcrumb[]; }> Event emitted when a breadcrumb is clicked, including the selected breadcrumb as well as the updated breadcrumbs after that breadcrumb is selected.
**[DEPRECATED]** breadcrumbSelected () => event: CustomEvent<LfBreadcrumb> This event is deprecated and will be removed in the next major release. It is replaced by breadcrumbClicked.
