MAJOR

Feburary 3, 2022

3.0.0: UI Elements HTML Attribute Names using snake_case, Unpublished Some Elements

Breaking Changes

  • Changed all UI elements' HTML attribute names from camelCase to snake_case.
  • <lf-login>:
    • Added new element with support for PKCE extension for Single-Page App.
    • Dropped support for login with Web App Oauth flow.
    • interface AccountEndpoints: Removed acsUrl; renamed signoutUrl to wsignoutUrl.
    • Renamed interface AccessTokenCredentials to AuthorizationCredentials.
    • Renamed attribute access_token to authorization_credentials.
  • <lf-breadcrumbs>:
    • Renamed attribute breadcrumb_options to breadcrumbs.
  • <lf-file-explorer>:
    • Renamed event selectedNodeChange to nodeSelected.
    • Added event openSelectedNodeAsync.
    • Added attribute focused_node.
  • <lf-folder-browser>:
    • Renamed event selectedNodeChange to nodeSelected.
    • Added event openSelectedNodeAsync.
  • <lf-tree>:
    • interface TreeNode: Removed level.
    • Removed event selectedNodeChange.
  • Unpublished the following elements:
    • <lf-login-buttom>
    • <lf-login-menu>
    • <lf-loader>
    • <lf-field>
    • <lf-field-group>
    • <lf-field-multivalue>
    • <lf-repository-selector>
    • <lf-new-folder-dialog>
    • <lf-popup>

New

  • <lf-field-container>: Added events dialogOpened and dialogClosed.
  • <lf-field-adhoc-container>: Added events dialogOpened and dialogClosed.
  • Added additional context to support issues in <lf-user-feedback> dialog.

Fixes

  • Fixed adhoc container freezing in lf-documentation.
  • Removed email in <lf-user-feedback>.
  • Fixed <lf-user-feedback> sending feedback to dev Database in production urls (eg: signin.laserfiche.com).
  • Fixed metadata bugs that dropdown field and token-picker don't work on initialization.

Links

  • CDN: https://lfxstatic.com/Site/laserfiche-ui-components/3.0.0/laserfiche-ui-components.js
  • NPM: https://v-dev-tfs/DefaultCollection/Integrations/_packaging?_a=package&feed=LFNPMPackages&view=overview&package=%40laserfiche%2Ftypes-laserfiche-ui-components&version=3.0.0&protocolType=Npm
  • Documentation: https://lfxstatic.com/Site/laserfiche-ui-components/3.0.0/index.html#/
