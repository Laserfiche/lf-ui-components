Overview

Web Component for a toolbar with inline buttons

Use Case

A toolbar of actions that can be performed on items in a list or tree

HTML Element

Attribute Value Description
displayed_options ToolbarOption[] A list of toolbar options that should be displayed outside of the dropdown
dropdown_options ToolbarOption[] A list of toolbar options that should be displayed in the dropdown

Event Value Description
optionSelected (event: CustomEvent<ToolbarOption>) => Promise<void> event raised when a toolbar option is selected

ToolbarOption Interface

  • name: string
  • disabled: boolean
  • icon?: string[] | string
  • children?: ToolbarOption[]
  • tag?: any (Object that contains data about the toolbar option)
