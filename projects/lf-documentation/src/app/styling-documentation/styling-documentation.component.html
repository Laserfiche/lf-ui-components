Styling Components

Links to Stylesheets:

Importing Stylesheets

You can use the stylesheets in one of two ways:

  1. A <link> element inside your <head> element in your index.html with the relevant url. (Recommended)
  2. An import at the top of your styles.css, styles.scss file, or equivalent.

Once this is included, all RWC components will be styled.

Example Code:
<link id="lf-stylesheet" href="https://lfxstatic.com/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-laserfiche-lite.css" rel="stylesheet" />
@import "https://lfxstatic.com/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-laserfiche-lite.css";

Styling Non-components

Once you have imported the stylesheet, you can also style simple elements that are not part of the components, for example: buttons and text fields. You do this by adding a class to the element. Some elements will also have subtypes/substyles which you will have to specify.

The class names follow the convention:
Common Elements: lf-element-name
Examples: lf-button, lf-checkbox

Element Subtypes: subtype-element-name
Examples: primary-button, sec-button 


  <button>Unstyled</button>
  <button class="lf-button primary-button">Primary</button>
  <button class="lf-button sec-button">Secondary</button>
  <button class="lf-button">No Subtype</button>
      

  <input value="Unstyled">
  <input class="lf-text-field" value="Styled">

Dynamically Changing Styles/Themes

This can only be done if you import the stylesheet in the index.html via a link element. Example implementations are provided below (styling not included). 


  <div>
    <button (click)="setThemeLaserfiche()">Laserfiche</button>
    <button (click)="setThemeMicrosoft()">Microsoft</button>
    <button (click)="removeTheme()">No Theme</button>
  </div>
        

  setThemeLaserfiche() {
    const stylesheet = document.getElementById('lf-stylesheet');
    stylesheet.setAttribute('href', 'https://lfxstatic.com/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-laserfiche-lite.css');
  }

  setThemeMicrosoft() {
    const stylesheet = document.getElementById('lf-stylesheet');
    stylesheet.setAttribute('href', 'https://lfxstatic.com/npm/@laserfiche/lf-ui-components@NPM_VERSION/cdn/lf-ms-office-lite.css');
  }

  removeTheme() {
    const stylesheet = document.getElementById('lf-stylesheet');
    stylesheet.setAttribute('href', '');
  }
        

  <select id="theme-selector" (change)="setTheme()">
    <option value="laserfiche">Laserfiche Theme</option>
    <option value="microsoft">Microsoft Theme</option>
  </select>
        

  setTheme() {
    const selectElement = (<HTMLSelectElement>document.getElementById('theme-selector'));
    const themeSelected = selectElement.options[selectElement.selectedIndex].value;
    if (themeSelected === 'laserfiche') {
      this.setThemeLaserfiche();
    }
    else if (themeSelected === 'microsoft') {
      this.setThemeMicrosoft();
    }
  }
