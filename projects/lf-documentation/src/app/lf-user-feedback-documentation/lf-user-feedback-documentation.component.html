Overview

(Laserfiche Cloud only) Custom element that opens a dialog to ask for product feedback or suggestions. Feedback and suggestions are automatically collected in Redshift upon clicking "Submit".

Use Case

The feedback button can be styled as a button or as an item in a dropdown. The element requires the name of the hosting module, the account ID, and user ID. No other customization is needed.

HTML Element

Attribute Value Description
hosting_module string Name of the product that the feedback is being given on. If your product already sends tracking events to Redshift, use the same module name. This will be used to filter submissions by product.
hosting_context (optional) string Any additional context you want to include (to aid in interpretting the feedback and suggestions)
user_id string Cloud user ID
account_id string Cloud account ID

Styling

To style the button, use Javascript to find the button by ID:

var feedbackButton = document.getElementById('lf-user-feedback');
if (feedbackButton) {
feedbackButton.style.color = 'red';
}
