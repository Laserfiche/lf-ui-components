Laserfiche UI Components

The Laserfiche UI Components project is a collection of reusable web components that are available as framework-agnostic custom elements. The UI components include out-of-the-box Laserfiche experiences (e.g. document view, search, and import) as well as lower-level base components (e.g. trees, fields, etc.).

The Laserfiche UI Components makes it easy for you to integrate Laserfiche experiences into your application with minimal code.

Who should use the Laserfiche UI Components?

The Laserfiche UI Components is intended for developers and integrators looking for an easy way to connect to Laserfiche in their web applications.
