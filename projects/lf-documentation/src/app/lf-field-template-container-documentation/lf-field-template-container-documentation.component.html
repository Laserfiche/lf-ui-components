Overview

Web Component to give the user the ability to select a template and edit fields within the chosen template.

Use Case

Display an template field container for the user to interact with

HTML Element

Event Value Description
fieldValuesChanged () => CustomEvent<boolean> Event emitted when any field values are changed. Emits whether template is valid or not without forcing validation.
templateSelectedChange () => CustomEvent<number> Event emitted when the template selected is changed. Emits the templateId of the newly selected template.

Method Value Description
initAsync (providers: LfFieldTemplateProviders, templateIdentifier?: number | string) => Promise<void> Function that initializes the template field container with given provider. Optionally initialize to a template with specific id or name
getTemplateValue () => TemplateValue | undefined accessor that returns the current TemplateValue which includes name and values
clearAsync () => Promise<void> clears the template and fields
forceValidation () => boolean function that returns whether all fields within container are valid
