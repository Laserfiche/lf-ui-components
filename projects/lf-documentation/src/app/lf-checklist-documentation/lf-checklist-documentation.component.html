lf-checklist output:

Overview

Web Component to give the user the ability to display a list of groups (each with a description, list of items (which have an icon, name, and optional checkbox), and a list of checkbox options) and allow the user to change the names and checkbox options before clicking the action (i.e. save, ok, etc.) button

Use Case

Upload to Laserfiche (Email and Attachments)

HTML Element

Attribute Value Description
action_button_text (optional) string Action button text
cancel_button_text (optional) string Cancel button text

Event Value Description
checklistChanged () => event: CustomEvent<Checklist[]> Event emitted when lf-checklist's checkbox or input data changes
actionClick () => event: CustomEvent<Checklist[]> Event emitted when the action button is clicked
cancelClick () => event: CustomEvent<Checklist[]> Event emitted when the cancel button is clicked

Method Value Description
initAsync (providers: LfChecklistProviders) => Promise<void> Function that allows dynamically triggering a checklist update (will call the ChecklistService function loadChecklistsAsync)
forceValidation () => boolean Function that triggers any validation messages and returns whether or not the checklists are valid
